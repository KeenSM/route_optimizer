### Route Optimizer CHANGELOG

All notable changes to this project will be documented in this file.

Status: [Released]



#### [1.0.1] - 2025.11.30

###### Added:

- Documentation updated to make the script more intuitive to use

###### Changes:

- Minor Bug fixes



#### [1.0.0] - 2025.11.20

###### Added:

- Memory allocation added for requests greater than 8 intermediates
- Minor route optimization added for requests greater than 8 intermediates
- Documentation created

###### Changes:

- Script can now handle 12 intermediate destinations
- Output now prints to a text file



#### [0.2.0] - 2025.11.16

###### Added:

- Matlab app created to serve as a smarter method for user input

###### Removed:

- Generate_prompts user function replaced by matlab app input



#### [0.1.1] - 2025.11.13

###### Added:

- Ability to select and calculate travel distance
- Traffic aware travel time added
- User functions implemented to clean up API calling from main script



#### [0.1.0] - 2025.11.12

###### Added:

- Mode of transport selector
- Functionality up to 11 intermediate destinations
- Run time improvements: Calculations done multiple orders of magnitude faster
